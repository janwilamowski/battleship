# BATTLESHIP

![Battleship](logo.png)

## Usage

```text
./battleship.py

```

## Controls

- Cursor keys/mouse: move crosshair
- Enter/left click: fire
- Escape: Quit
- N: New game

## Download

Find and download the directory for your system in the [dist folder](./dist/). It contains an executable and all required resources.

## Requirements

- Python 3
- PyGame
- pgu
